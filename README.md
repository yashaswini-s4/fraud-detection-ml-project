# Fraud Detection using Machine Learning

## Objective
To detect fraudulent financial transactions using machine learning techniques.

## Models Used
- Random Forest
- XGBoost
- LightGBM

## Key Features
- Data Cleaning
- Feature Engineering
- Model Comparison
- Imbalance Handling

## Results
- High recall achieved for fraud detection
- ROC-AUC score used for evaluation

## Tools Used
Python, Pandas, Scikit-learn, LightGBM

The dataset is too large to upload to GitHub.
